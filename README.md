# Translate - AI 智能翻译

英译中网页翻译器，由 DeepSeek AI 驱动翻译，Google Cloud TTS 提供朗读。

## ✨ 功能

- 🔤 **智能翻译** — DeepSeek AI 流式翻译，支持 ⌘+Enter 快捷键
- 📖 **生词本** — 收藏单词，支持搜索和管理
- 🔊 **朗读** — Google Cloud TTS Wavenet 高品质语音（中/英）
- 🌙 **深色/浅色主题** — 一键切换

## 🛠️ 技术栈

- Vite + React 18
- Tailwind CSS v3 + shadcn/ui
- DeepSeek API（翻译）
- Google Cloud TTS API（朗读）

## 🚀 快速开始

```bash
npm install
npm run dev
```

打开 http://localhost:5173，在设置中配置 API Key 即可使用。
